# Story 001.002: PageShell, BioLine, AntipatternsStrip, CurrentlyStrip

## Description

Build the persistent chrome that appears on every page: `<PageShell>` (top bar + nav + "currently" data strip), `<BioLine>` (locked bio copy), and `<AntipatternsStrip>` (the "I don't do" list). These components are the site's front door ‚Äî they render before any Fieldwork or Postcard content and are the first things a visitor sees.

## Acceptance Criteria

- [ ] AC-001: `<PageShell>` wraps all pages via root `layout.tsx`, renders top bar with stamp (left) + site wordmark + nav, and renders footer with antipatterns strip
- [ ] AC-002: `<BioLine>` renders the locked copy verbatim: *"Kentucky-raised, London-based, accidentally Canadian-sounding. I build AI for regulated industries by day and argue with it by night."* ‚Äî copy lives in `src/lib/content/site.ts` as a constant, not hardcoded in the component
- [ ] AC-003: `<AntipatternsStrip>` renders the full locked list as-drafted in the architecture: *ROI calculators ¬∑ AI-hype decks ¬∑ five-minute reads ¬∑ thought-leadership videos ¬∑ "here are 7 things I learned" ¬∑ "I asked ChatGPT to‚Ä¶" posts ¬∑ robot illustrations ¬∑ newsletter pop-ups ¬∑ posts that resolve cleanly at the end*. Copy lives in `src/lib/content/antipatterns.ts`.
- [ ] AC-004: `<CurrentlyStrip>` renders the JetBrains Mono data strip with the "currently" fragment + site stats (fieldwork count, postcards count, changed-my-mind count, last updated date). Values are passed as props; derivation lives in the content pipeline (Story 001.003) ‚Äî for this story use hardcoded placeholders that the content pipeline will later replace.
- [ ] AC-005: Nav includes links to `/` (home via stamp), `/fieldwork`, `/postcards`, `/archive`, `/now`, `/argue`. Pages don't exist yet ‚Äî links are allowed to 404; they'll resolve in downstream stories.
- [ ] AC-006: Visual verification against mock header in `~/Documents/bines-ai-brainstorm/stamp-final.html`: stamp size, wordmark styling with ruby dot separator, nav spacing, "currently" line styling match.
- [ ] AC-007: Keyboard-navigable ‚Äî tab order is sensible (skip link to main content, nav links reachable).
- [ ] AC-008: `prefers-reduced-motion`: any hover/focus transitions gated behind the media query.

## Technical Tasks

1. Create `src/lib/content/site.ts` with `bioLine` constant, nav config, and default `currentlyLine` fallback
2. Create `src/lib/content/antipatterns.ts` with the locked `antipatterns` string array
3. Build `<PageShell>` (server component) ‚Äî renders top bar + `<CurrentlyStrip>` + `{children}` main + footer with `<AntipatternsStrip>`
4. Build `<BioLine>` (server) ‚Äî takes no props, renders bio constant
5. Build `<AntipatternsStrip>` (server) ‚Äî maps over antipatterns list
6. Build `<CurrentlyStrip>` (server) ‚Äî props: `current: string`, `counts: { fieldwork, postcards, changedMyMind }`, `updated: Date`
7. Build `<Stamp>` (server) ‚Äî takes `colour?: ColourToken` and `size?: 32|56|96|240`, returns inline SVG (not `<img>`) so CSS `currentColor` works. Reuse paths from `public/stamp.svg`.
8. Build `<Wordmark>` (server) ‚Äî renders "bines" in Fraunces 900 + ruby dot + "ai" ‚Äî matches mock header
9. Update root `layout.tsx` to wrap children with `<PageShell>`
