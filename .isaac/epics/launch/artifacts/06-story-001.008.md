# Story 001.008: /now and /taste pages

## Description

Build the two single-file MDX surfaces that aren't Fieldwork or Postcards: `/now` (monthly update — currently obsessed / failing / rereading) and `/taste` (3-5 current reads / watches / listens). Both update by editing a single MDX file; git log serves as their archive.

## Acceptance Criteria

- [ ] AC-001: `/now` renders `content/now.mdx` with frontmatter showing `updated` date and `currently` line, plus the body paragraph
- [ ] AC-002: `/taste` renders `content/taste.mdx` with 3-5 items, each optionally linked
- [ ] AC-003: Both pages respect the editorial-maximalist aesthetic: generous spacing, Fraunces body type, one jewel accent per page
- [ ] AC-004: The `currently` line from `/now` feeds into `<CurrentlyStrip>` in `<PageShell>` (replaces the hardcoded placeholder from 001.002)
- [ ] AC-005: Nav links `/now` resolves (already in nav from 001.002)
- [ ] AC-006: Both pages include a "last updated" date visibly rendered — honest signal that these are live documents
- [ ] AC-007: Both MDX files include seed content for launch: `/now` = a real current-state paragraph (Maria to provide, or placeholder that she edits before launch); `/taste` = 3 seed items

## Technical Tasks

1. Create `content/now.mdx` with initial frontmatter + body
2. Create `content/taste.mdx` with 3 seed items (`{ title, by, link?, note? }[]`)
3. Build `src/app/now/page.tsx` — server component using `getNow()` loader
4. Build `src/app/taste/page.tsx` — server component using `getTaste()` loader
5. Build `src/components/NowBlock.tsx` — renders now content
6. Build `src/components/TasteShelf.tsx` — renders taste items
7. Wire `<CurrentlyStrip>` in `<PageShell>` to pull from `getNow()` — update 001.002's placeholder

## Files

- **CREATE**: `content/now.mdx`, `content/taste.mdx`, `src/app/now/page.tsx`, `src/app/taste/page.tsx`, `src/components/NowBlock.tsx`, `src/components/TasteShelf.tsx`
- **MODIFY**: `src/components/CurrentlyStrip.tsx` (accept prop from live data instead of placeholder), `src/components/PageShell.tsx` (wire the real data)

## Scope rules

- Don't build an admin UI for updating these pages. Maria edits MDX directly.
- Don't add revision history UI (git log is the archive).
- Don't add email notifications on /now update. Out of scope for v1.
