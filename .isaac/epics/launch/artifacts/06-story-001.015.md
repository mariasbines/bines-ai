# Story 001.015: RSS + sitemap + robots + OG defaults

## Description

Build the discoverability scaffolding: a combined RSS feed of Fieldwork + Postcards, a dynamic sitemap, a robots.txt, a default OG image, and per-page metadata (title/description/og-image). These earn their keep for search engines, social sharing, and anyone who subscribes via RSS.

## Acceptance Criteria

- [ ] AC-001: `/rss.xml` returns a valid RSS 2.0 feed with: site title, description, self-link, `<item>` per Fieldwork piece + per Postcard, sorted by `pubDate` descending, with proper `<content:encoded>` HTML of the piece body
- [ ] AC-002: `/sitemap.xml` lists: `/`, `/fieldwork/[slug]` for all pieces (in-rotation + retired + changed-my-mind), `/postcards`, `/postcards/[number]`, `/archive`, `/now`, `/taste`, `/argue`, `/changed-my-mind/[slug]` тАФ with appropriate `<lastmod>` dates
- [ ] AC-003: `/robots.txt` allows all crawlers except `/api/` paths
- [ ] AC-004: Default OG image at `public/og-default.png` тАФ 1200├Ч630, uses the stamp + bio line + a jewel anchor, matches editorial-maximalist aesthetic
- [ ] AC-005: Per-page metadata via Next `generateMetadata()`: each Fieldwork piece has title = piece title, description = excerpt, `openGraph.images = [ogImageForPiece(slug)]`
- [ ] AC-006: Per-Fieldwork OG image: dynamically generated via Next's `opengraph-image.tsx` route using `ImageResponse` тАФ title + accent-colour block + stamp. Static for v1 (generated at build), can become per-request in v2 if desired.
- [ ] AC-007: Postcards get a simpler OG image: "POSTCARD #NNN" + short excerpt in cream-on-ink
- [ ] AC-008: RSS feed link discoverable via `<link rel="alternate" type="application/rss+xml" ...>` in root layout head

## Technical Tasks

1. Build `src/app/rss.xml/route.ts` using the `feed` package тАФ composes Fieldwork + Postcards
2. Build `src/app/sitemap.ts` (Next dynamic sitemap) тАФ enumerates all routes
3. Build `src/app/robots.ts` (Next dynamic robots)
4. Create `public/og-default.png` тАФ design in Figma or generate via headless render; check into repo for v1
5. Build `src/app/fieldwork/[slug]/opengraph-image.tsx` тАФ `ImageResponse` per-piece OG
6. Build `src/app/postcards/[number]/opengraph-image.tsx` тАФ per-postcard OG
7. Update each page's `generateMetadata()` to include title, description, openGraph
8. Add `<link rel="alternate">` in root layout `<head>`

## Scope rules

- Do NOT split Fieldwork and Postcards into separate RSS feeds in v1 (one combined feed; separate feeds are a v2 decision)
- Do NOT add email subscription тАФ antipatterns forbid newsletter pop-ups
- Do NOT add "related posts" sections
- Do NOT add breadcrumbs тАФ site is shallow enough that they're noise
