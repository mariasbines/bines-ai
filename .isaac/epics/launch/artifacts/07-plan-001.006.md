# Implementation Plan: 001.006 — Postcard rendering (card + article + routes)

## Overview

Builds the Postcards reading surface: `<PostcardCard>`, `<PostcardArticle>`, `/postcards` index, and `/postcards/[number]` dynamic route. Chuck Lorre vanity-card register — numbered, brief, signed off, no metadata strip. Visually quieter than Fieldwork; jewels still rotate but keep a lighter touch. Homepage gets a "recent Postcards" section under the Fieldwork cards.

## Design decisions locked upfront

### 1. Numbering format — 3-digit zero-pad via `src/lib/utils/number.ts`

Per AC-005 + AC-004: `/postcards/001`, `/postcards/047`, `/postcards/123`. URL segment and visible header both use `padStart(3, '0')`.

### 2. Accent rotation

Postcards also rotate jewels (AC-006) but more quietly — per-card accent applied only to the `POSTCARD #NNN` label, not to the body. Use the same `accentFor(piece)` helper from 001.004; fallback `number % 5`.

### 3. Shape: no metadata strip

Unlike Fieldwork, postcards don't render `<Metadata>`. Per AC-001: number, body, date line, signoff. Signoff format: `maria · dd mmm` (per state-of-play Postcard #001 draft).

### 4. Index page layout

Reverse-chronological by number (newest first). No pagination in v1 per AC-003 ("pagination ready but not required"). We render a simple stack with ample vertical spacing; each entry gets a number header + body + date.

### 5. Homepage integration

Per "Files to Modify" list, `src/app/page.tsx` gets a "recent postcards" section below the Fieldwork grid. Last 3 postcards. Graceful empty-state when no postcards exist.

### 6. Reuse `<MdxBody>`

Postcard bodies are short MDX (50-200 words). Reuse `<MdxBody>` — same pipeline as Fieldwork, same component overrides available (though typical postcards won't need `<PullQuote>`).

### 7. `params` shape

Route `/postcards/[number]`: `params: Promise<{ number: string }>`. Parse to int before lookup; `notFound()` if NaN or not found.

## Implementation Steps

### Step 1 · `src/lib/utils/number.ts`

```typescript
/** Zero-pad a positive integer to at least 3 digits. */
export function padNumber(n: number, width: number = 3): string {
  return n.toString().padStart(width, '0');
}
```

### Step 2 · `src/components/PostcardCard.tsx` (server)

```typescript
import Link from 'next/link';
import { format } from 'date-fns';
import type { Postcard } from '@/lib/content/types';
import { accentFor, accentVar } from '@/lib/design/accent';
import { padNumber } from '@/lib/utils/number';
import { MdxBody } from './MdxBody';

interface PostcardCardProps {
  postcard: Postcard;
  linkTitle?: boolean;
}

const fmtDateShort = (iso: string) => format(new Date(`${iso}T00:00:00Z`), 'd MMM').toLowerCase();

export function PostcardCard({ postcard, linkTitle = true }: PostcardCardProps) {
  const { number, published } = postcard.frontmatter;
  const numPadded = padNumber(number);
  const accent = accentFor({ frontmatter: { id: number, accent: postcard.frontmatter.accent } });
  const header = (
    <span className="font-mono text-xs uppercase tracking-[0.14em] text-accent">
      postcard #{numPadded}
    </span>
  );

  return (
    <article
      className="border-t border-ink/15 pt-6 pb-8"
      style={{ ['--color-accent' as string]: accentVar(accent) } as React.CSSProperties}
    >
      <header className="mb-4">
        {linkTitle ? (
          <Link href={`/postcards/${numPadded}`} className="hover:opacity-80 transition-opacity motion-reduce:transition-none">
            {header}
          </Link>
        ) : header}
      </header>

      <div className="font-serif text-base leading-relaxed [&_p]:my-3 [&_em]:italic">
        <MdxBody source={postcard.body} />
      </div>

      <p className="mt-5 font-mono text-xs text-ink/60 italic">
        maria · {fmtDateShort(published)}
      </p>
    </article>
  );
}
```

**Note on `accentFor` input shape:** it expects `{ frontmatter: { id, accent? } }`. Postcards have `number` not `id` — we pass `{ id: number }` shape to reuse. Could generalise the helper, but a minimal cast at the callsite is simpler and keeps 001.004's signature stable.

### Step 5 · `src/app/postcards/[number]/page.tsx` (SSG single)

```typescript
import { notFound } from 'next/navigation';
import { getAllPostcards, getPostcardByNumber } from '@/lib/content/postcards';
import { PostcardArticle } from '@/components/PostcardArticle';
import { padNumber } from '@/lib/utils/number';

export async function generateStaticParams() {
  const postcards = await getAllPostcards();
  return postcards.map((pc) => ({ number: padNumber(pc.frontmatter.number) }));
}

interface PageProps {
  params: Promise<{ number: string }>;
}

export default async function PostcardPage({ params }: PageProps) {
  const { number: numberParam } = await params;
  const n = Number(numberParam);
  if (!Number.isInteger(n) || n <= 0) notFound();
  const pc = await getPostcardByNumber(n);
  if (!pc) notFound();
  return <PostcardArticle postcard={pc} />;
}
```

```typescript
import { BioLine } from '@/components/BioLine';
import { FieldworkCard } from '@/components/FieldworkCard';
import { PostcardCard } from '@/components/PostcardCard';
import { getAllFieldwork } from '@/lib/content/fieldwork';

import Link from 'next/link';

export default async function Home() {
  const [fieldwork, postcards] = await Promise.all([

    getAllPostcards(),

  return (

      <BioLine />

      {fieldwork.length > 0 ? (
        <section aria-label="Fieldwork in rotation" className="grid gap-8 sm:grid-cols-2">

      {recentPostcards.length > 0 ? (
        <section aria-label="Recent postcards">
          <header className="flex items-baseline justify-between mb-4">
            <h2 className="font-serif font-black text-2xl tracking-tight">Recent postcards</h2>
```

```typescript
import { padNumber } from '../number';

describe('padNumber', () => {
  it('zero-pads to 3 digits by default', () => {
    expect(padNumber(1)).toBe('001');
    expect(padNumber(47)).toBe('047');
```
