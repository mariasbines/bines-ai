# Story 001.004: Fieldwork rendering — card + article + route

## Description

Build the core Fieldwork reading experience end-to-end: `<FieldworkCard>` (the data-strip card used on the homepage), `<FieldworkArticle>` (the full piece renderer), and the `/fieldwork/[slug]` dynamic route. Per-piece jewel anchor pulled from frontmatter drives the accent colour. Does NOT yet integrate video (that's 001.013) or push-back (that's 001.014) — those are handled as extension points here.

## Acceptance Criteria

- [ ] AC-001: `<FieldworkCard>` renders exactly matching the spec in the mood board: top strip `FIELDWORK NN · {status}`, title in Fraunces heavy, metadata block (published, media, pushback, change-my-mind, tags) in JetBrains Mono, CTA buttons `[ watch ] [ read ] [ push back ]` styled per accent jewel
- [ ] AC-002: `<FieldworkArticle>` renders: hero area (placeholder for video loop until 001.013), title (Fraunces 900 clamp 34-54px), byline strip, MDX body with component overrides for `<PullQuote>`, change-my-mind footer slot
- [ ] AC-003: `/fieldwork/[slug]` dynamic route uses `generateStaticParams()` to SSG all in-rotation pieces; retired pieces are SSG too but marked with a small *"retired — see archive"* banner
- [ ] AC-004: Per-piece accent colour: frontmatter may include `accent: 'emerald' | 'sapphire' | 'ruby' | 'topaz' | 'amethyst'`; if absent, derive deterministically from `id % 5`. Accent is applied via CSS custom property `--color-accent` on the article root.
- [ ] AC-005: MDX supports these components: `<PullQuote>` (the big italic Fraunces pull-quote style), `<Aside>` (parenthetical note), `<FigureCaption>` (typography for captions)
- [ ] AC-006: Homepage (`/`) renders `<FieldworkCard>` for each `in-rotation` piece in a grid or column (lock layout in this story; 2-column on desktop, single column on mobile)
- [ ] AC-007: Metadata strip uses `date-fns` to format dates consistently ("14 Apr 2026")
- [ ] AC-008: Response time: Fieldwork page LCP under 2.5s on throttled connection (pre-video; video LCP deferred to 001.013)
- [ ] AC-009: Reading view has proper `<h1>` (title), semantic `<article>`, `<time datetime>` on the published date

## Technical Tasks

1. Build `src/components/FieldworkCard.tsx` (server) — takes `piece: Fieldwork` prop, renders card per mood-board spec
2. Build `src/components/FieldworkArticle.tsx` (server) — takes `piece: Fieldwork`, renders full article
3. Build `src/components/Metadata.tsx` (server) — reusable data strip used inside cards and articles
4. Build `src/components/PullQuote.tsx`, `src/components/Aside.tsx`, `src/components/FigureCaption.tsx` — MDX components (server)
5. Build `src/components/VideoLoopPlaceholder.tsx` — a server component that renders a solid-colour rectangle at the correct aspect ratio, replaced by actual `<VideoLoop>` in 001.013
6. Build `src/app/fieldwork/[slug]/page.tsx` with `generateStaticParams` from `getAllFieldwork()`
7. Update `src/app/page.tsx` to fetch `getAllFieldwork({ status: 'in-rotation' })` and render cards
8. Build `src/lib/design/accent.ts` — map accent token to CSS variable, deterministic fallback from id
9. Wire Tailwind utilities for `text-accent`, `border-accent`, `bg-accent` to consume `--color-accent`
