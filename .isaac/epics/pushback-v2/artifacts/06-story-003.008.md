# Story 003.008: [ argue with this ] CTA on FieldworkCardCtas (Phase C)

**Epic:** `pushback-v2`
**Phase:** C — public surfaces (release-blocker bundle)

## Description

Replaces the hidden `[ push back ]` element in `<FieldworkCardCtas>` with a visible `<Link href="/argue?from=<slug>">[ argue with this ]</Link>`. Same shape as the existing `[ watch ]` and `[ read ]` CTAs — lowercase, square brackets, font-mono. Slug is already available in scope via `piece.frontmatter.slug`. No state, no client-side handler — it's a regular Next link.

This story is intentionally small. The CTA copy was locked in concept Q4 (`[ argue with this ]`); the route handling for `?from=<slug>` shipped in 003.002. This story is the visible CTA on every Fieldwork index card. It must merge alongside or after 003.007 (the count badge sits next to it visually; if the CTA ships without the badge ready, the index card looks half-built).
